Jordar Food Limited
A web application developed for Jordar Food Limited to support their organic produce and farming operations.

Project Overview
This project is a dedicated web platform designed to streamline the operations of an organic produce and farming business. It provides a professional online presence to showcase farm products and connect with customers.

Technologies Used
Framework: Django

Styling: Tailwind CSS

Features
Homepage for Jordar Food Limited

Showcase of farm produce and business information
